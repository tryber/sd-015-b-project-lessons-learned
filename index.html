Sobre Meu Aprendizado

Aprendizado cheio de ansiedade, mas com a ajuda das colegas, principalmente da Laiza!

Após 2 semanas de curso sinto que meus conhecimentos relacionados aos temas estudados são em muito superiores ao que imagiinava ser possível.

Claro que este processo teve diversos momentos de desespero, de agonia, de muita ansiedade e sensação de impostora. Porém o grupo de mulheres e as trocas que tivemos foram importantíssimas para me fortalecer.

Uma passagem deste momento de aprendizado que destaco, até o momento, foi quando finalmente me senti confortável trabalhando no GitHub, praticamente me sentindo dona e proprietária dos comandos git status, git add, git commit, git push, git clone, etc.
