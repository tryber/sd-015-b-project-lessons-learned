Minha experiência de aprendizado na Trybe

Aprendemos desde o início a usar o terminal, junto de algumas peculiaridades do Linux. O domínio do terminal foi fundamental para aprendermos a utilizar o Git. Após o aprendizado de versionamento no Git, aprendemos a trabalhar na interface do Github, fazendo commits, pull requests, code review e merge. Após isso, prosseguimos para o HTML básico, CSS, posicionamento de CSS e o HTML semântico.

Minhas expectativas foram totalmente superadas, desde a primeira semana. Não esperava que íamos lidar com terminal de primeira, e aprender tanta coisa sobre o Linux, Git e sobre a Internet.
