Como é o aprendizado na Trybe

Na Trybe o conteúdo é intenso, desde o primeiro dia de conteúdo conseguimos observar como funciona a metodologia do curso. O primeiro conteúdo foram os Comandos de terminal, logo depois já veio Git e GitHub, num primeiro momento tomei um susto pois é tudo novo, quando começamos HTML e CSS o conteúdo de Git e GitHub fica mais tranquilo, pois como diz o querido Isaac "Git é todo dia, Git é Lindo, Git é Tudo".Uma semana na Trybe parace um mês. Dessa forma seguimos adiante em busca do nosso objetivo, vamos com tudo!
